# Axiom

Proyecto GameTracker para Jovenes CreaTIvos 2025, desarrollado por Joshua Berdugo

# Qué es ***Axiom***?

**Axiom** es una aplicación web que permite a los usuarios llevar un registro organizado de los videojuegos que poseen, han jugado o desean jugar. 

La plataforma incluye un ***sistema de reseñas***, lo que facilita evaluar cada juego y construir una biblioteca personal completamente administrable.

El proyecto combina un **frontend** en *React*, un **backend** en *Node.js* + *Express* y la base de datos en **MongoDB** suministrada por **JC**, creando una experiencia simple, rápida y moderna. 
Axiom está diseñado como una herramienta práctica para jugadores que buscan gestionar su colección y guardar información relevante de cada título en un solo lugar.
